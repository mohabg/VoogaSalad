Provides the necessary classes, concrete and abstract, to build the toolbar in the authoring environment. These classes are: AbstractManu, AddNewLevelMenu, AuthoringMenubarCreator, BackMenu, FileMenu, PlayMenu, SaveGameMenu. AbstractMenu is the abstract class that AddNewLevelMenu, BackMenu, FileMenu, PlayMenu, and SaveGameMenu all extend. ItemWindow is the essential class that creates each tab and adds them to the window. It pulls some of its information from ItemWindowData, where the data for SpriteImages, ItemPaths, and tab titles is determined. Within each tab there is a TilePane.
